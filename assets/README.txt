
Risk-Themed Overlay Assets
==========================

Files:
- map_texture.jpg         : Background (1920x1080 dark parchment + warm glows)
- paper_texture.png       : Subtle paper texture tile (512x512)
- risk_style_logo.png     : Stylized "Barry Risk" wordmark (not official Risk logo)
- flag_icon.png           : For message marker
- shield.png              : Mod badge
- crown.png               : VIP badge
- star.png                : Subscriber badge
- globe.png               : Broadcaster badge
- tank_icon.png           : Extra icon for fun / future use

How to use:
1) Commit this /assets folder to the root of your GitHub repo.
2) In your index.html CSS, point backgrounds/icons to 'assets/<file>'.
3) Example URL for Risk theme in OBS:
   https://<username>.github.io/<repo>/?channel=barry_risk&theme=risk&showheader=1&compact=1&mention=1&v=6
